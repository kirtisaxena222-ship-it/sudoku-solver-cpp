# Sudoku Solver (C++)

This project solves a 9x9 Sudoku puzzle using the Backtracking Algorithm.

## Features
- Solves any valid Sudoku puzzle
- Uses recursion and backtracking
- Displays solved grid in console

## Concepts Used
- Recursion
- Backtracking
- 2D Arrays
- Algorithm design
